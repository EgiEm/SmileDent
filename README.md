# SmileDent — Dr. Besnik Emini

A premium, highly interactive single-page landing website designed and developed for **SmileDent**, a professional dental clinic led by Dr. Besnik Emini in Gjilan, Kosovo. 

The website is optimized for conversion, providing a smooth user experience for patients to explore dental services, view treatment results, and request appointments.

Live Demo: [https://egiem.github.io/SmileDent/](https://egiem.github.io/SmileDent/)

---

## 🛠️ Key Features

- **Before & After Slider:** An interactive, touch-friendly image comparison component built from scratch using vanilla JavaScript to showcase treatment outcomes.
- **Appointment Request Flow:** Interactive modal and page forms with live validation and success feedback, designed to capture booking requests.
- **Fully Responsive Layout:** Mobile-first approach with custom CSS grid layouts and a sliding navigation drawer for mobile viewports.
- **Modern Medical Aesthetic:** Calm color palette using teal and indigo pastel accents, clean typography (Outfit & Plus Jakarta Sans), and micro-animations.
- **Business Details & Map:** Clear schedule layout, clickable quick-dial phone numbers, direct links to social profiles, and an embedded Google Map pointing exactly to the clinic's location.

```
SmileDent/
├── index.html       # Semantic HTML5 document & layout structure
├── style.css        # Custom CSS styles, variables, transitions, and media queries
```

---

## 💻 Tech Stack

- **Markup:** HTML5 (Semantic elements)
- **Styling:** CSS3 (Variables, Flexbox, Grid, custom keyframes)
- **Interactivity:** Vanilla JavaScript (ES6+)
- **Typography & Icons:** Google Fonts (Outfit, Plus Jakarta Sans), FontAwesome 6 (Free)

## 🚀 Local Setup

To preview the website locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/EgiEm/SmileDent.git
   ```
2. Navigate into the directory:
   ```bash
   cd SmileDent
   ```
3. Open `index.html` in your web browser, or serve it using a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   ```
